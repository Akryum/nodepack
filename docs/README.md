## Quick start

Install the Command-Line Interface:

``` bash
npm install -g @nodepack/cli
# OR
yarn global add @nodepack/cli
```

Create a project:

``` bash
nodepack create my-project
```

Run it in development mode:

```bash
cd my-project
nodepack
```
